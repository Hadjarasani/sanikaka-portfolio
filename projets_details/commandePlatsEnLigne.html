🛠 Technologies utilisées : Flutter, Firebase, API REST, GetX, PayPal Sandbox.

📌 Objectif : Créer une application mobile permettant aux utilisateurs de commander des plats africains et européens avec un processus de commande simple et intuitif.

✅ Travail réalisé :

Backend :
  • Configuration et gestion de l’authentification sécurisée avec Firebase Authentication.
  • Implémentation du stockage des utilisateurs et des commandes avec Firebase Firestore.
  • Intégration d’une API REST pour récupérer les données des plats disponibles.
  • Gestion des transactions via PayPal Sandbox pour les paiements en ligne.
Frontend :
  • Développement de l’interface utilisateur en Flutter.
  • Utilisation du design pattern Bloc pour la gestion des états (affichage/masquage du mot de passe).
  • Intégration du package GetX pour la gestion des états et la navigation entre les écrans.
  • Création de composants réutilisables (champs de texte, boutons) pour une meilleure maintenabilité.

🎯 Expérience et apprentissages :

Ce projet m'a permis d'acquérir des compétences en développement mobile avec Flutter, en gestion des états avec GetX et en intégration d’API REST pour interagir avec une base de données Firebase.
