🛠 Technologies utilisées : PostgreSQL + PostGIS, Geoserver, React.js, HTML5, CSS3, React Native, OpenLayers, OpenStreetMap (OSM).

📌 Objectif : Développer une application collaborative permettant la visualisation des campus de l’Université d’Orléans et l’ajout interactif de bâtiments avec leurs services et publics cibles.

✅ Travail réalisé :

  • Mise en place d’une base de données spatiale sous PostGIS.
  • Écriture de requêtes SQL avancées pour la gestion des données.
  • Intégration de GeoServer pour exposer les données spatiales sous forme de services web cartographiques.
  • Création d’une interface interactive sous React Native (mobile) et React.js / HTML5 pour le web.
  • Affichage de la carte avec OpenLayers et OSM

🚀 Fonctionnalitées principales :

  • Visualisation des bâtiments sur une carte OSM avec affichage des informations associées.
  • Filtres dynamiques pour sélectionner les bâtiments en fonction de certaines informations.
  • Ajout et Modification des coordonnées GPS en cliquant directement sur la carte.
  • Extraction des données sous format CSV et GeoJSON, avec possibilité de filtrer

🎯 Expérience et apprentissages :

Ce projet m’a permis d'acquérir des compétences en systèmes d’information géographiques avec PostGIS et GeoServer, gérer, manipuler et extraire des données spatiales.
